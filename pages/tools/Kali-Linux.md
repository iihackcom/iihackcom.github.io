# Kali Linux

Kali Linux 基于[Debian](https://www.debian.org/) 发行，旨在进行高级渗透测试和安全审核。Kali Linux 由领先的信息安全培训公司 Offensive Security 开发，资助和维护，也是OSCP认证首选操作系统。
